# 🚀 FYP Automation System (Web + Mobile)

This repository contains a full-stack Final Year Project with:

- 🌐 WEBSITE (Web Application) – Next.js
- 📱 mobileFyp (Mobile Application) – React Native

Both applications are part of the same system and share the same backend, roles, and features.

---

# 📁 Project Structure

FYP/
│
├── WEBSITE/        → Web Application (Next.js)
├── mobileFyp/      → Mobile Application (React Native)

# 🌟 System Overview

The system is a **unified FYP management platform** designed for:

- Students
- Teachers / Supervisors
- Committee Members
- Admins

It supports both **web and mobile access** with shared functionality.

---

# 🚀 How to Run Project

## 🌐 Website

cd WEBSITE
npm install
npm run dev

Build production:
npm run build
npm start

---

## 📱 Mobile App

cd mobileFyp
npm install
npm start

Run Android:
npm run android

Run iOS:
bundle install
bundle exec pod install
npm run ios
